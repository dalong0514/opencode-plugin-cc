项目已有 GitHub 远程(`dalong0514/opencode-plugin-cc`)。注意:最新提交 `95abd88` 还没推送,走 GitHub 安装前需先 `git push`(需要我帮你推吗?)。

### 方式一:通过 GitHub 安装(推荐)

**新主机前置条件:**

```bash
# 1. Node >= 18.18(插件脚本运行时)
node --version

# 2. opencode CLI
npm install -g opencode-ai     # 或 bun add -g opencode-ai

# 3. 配置 provider(二选一)
opencode auth login            # 配 DeepSeek 等 API key
# 或不配,直接用免费的 opencode/* 系列模型
```

**安装插件**(Claude Code 内执行):

```
/plugin marketplace add dalong0514/opencode-plugin-cc
/plugin install opencode@opencode-cc
```

然后重启 Claude Code(或 `/reload-plugins`),跑 `/opencode:setup` 验证就绪。

也可以在终端一步完成:

```bash
claude plugin marketplace add dalong0514/opencode-plugin-cc
claude plugin install opencode@opencode-cc
```

⚠️ 如果仓库是 **private**,新主机需要先配好 GitHub 访问凭据(SSH key 或 `gh auth login`),marketplace add 本质是 git clone。

### 方式二:本地目录安装(无网/内网环境)

把项目目录整个复制(或 `git clone`)到目标主机任意路径,然后:

```
/plugin marketplace add D:\path\to\opencode-plugin-cc
/plugin install opencode@opencode-cc
```

### 后续更新

其他主机上更新到新版本:

```bash
claude plugin marketplace update opencode-cc
claude plugin uninstall opencode@opencode-cc && claude plugin install opencode@opencode-cc
```

(如果发布时 bump 了 `version`,直接 `claude plugin update opencode@opencode-cc` 即可;同版本号改动必须卸载重装才会刷新缓存。)

装好后即可在任意项目里:**"指定使用opencode的模型 deepseek/deepseek-v4-pro(--effort high)来完成如下任务:…"** 或 `/opencode:rescue --model … <任务>`。
